HTML
Hyper Text Markup Language

Hyper Text - 우리가 흔히 사용하는 링크를 얘기함
         	문서 안에 다른 문서로 연결되는 링크를 집어 넣어 웹상에 존재하는 여러 문서를 이동
  		텍스트, 그림, 소리 등을 하이퍼링크를 전달
   		
Markup Language - mark(태그)로 둘러 싸인 언어
                 		태그는 교정부호, 의미를 부여
      			태그 각각에 의미가 있음

앞에 있는 태그는 여는 태그 - 오프닝 태그
뒤에 있는 태그는 닫는 태그 - 클로징 태그
태그 사이에 존재하는 문장 - 콘텐츠
태그와 콘텐츠를 모아서 한번에 - 엘리먼트

태그 검색할때 html oo 태그 mdn 으로 검색

태그의 종류

h태그(1~6) - 제목 태그
		h1은 전체 제목을 표현할때 사용 - 보통 한 번만 사용
		그리고h2~h6은 부제, 가제 표현 --> 순서대로 사용하는 것 권장

div 태그 - 레이아웃 구성 때 사용되는태그
	     가장 남용되는 태그
	     p태그 처럼 한 줄 차지  (Q. 한줄을 차지하는 것과 콘텐츠 만큼 차지한다는 것이 어떤의미?)
	     
시멘틱 태그 - div 남발 막기 위한 태그
                  div의 각 영역들을 나눠 표시

ul - 순서가 없는 리스트
ol - 순서가 있는 리스트



속성 - 태그로만 정보 전달이 힘들때 사용
	오프닝 태그 뒤에 작성
